📥 Step 1: Download & Install ALL 3 APKs

⬇️ Termux ⬇️ Termux:API ⬇️ Termux:Boot

Install all 3, then open each app once.

🔐 Step 2: Grant Permissions

  • Open Termux → type: termux-setup-storage → Allow
  • Settings → Apps → Termux → Permissions → Enable ALL
  • Settings → Apps → Termux:API → Permissions → Enable ALL

⚡ Step 3: Copy & Paste In Termux

Paste in Termux → Enter → Wait 3-4 mins → Bot LIVE
